# Fotos del Equipo

## Ubicación de las fotos

Las fotos de los miembros del equipo deben colocarse en esta carpeta:
`C:\xampp\htdocs\tutorial laravel\sitio-facturacion\public\images\team\`

## Nombres de archivo requeridos

Coloca las fotos con los siguientes nombres exactos:

1. **miguel-ayala.jpg** - Foto de Ing. Miguel Ángel Ayala (Full Stack Developer)
2. **carlos-posada.jpg** - Foto de Ing. Carlos Humberto Posada (Backend Developer)
3. **gabo.jpg** - Foto de Gabo (Product Owner)

## Especificaciones de las fotos

- **Tamaño recomendado**: 200x200 píxeles (o cualquier tamaño cuadrado)
- **Formato**: JPG o PNG
- **Peso máximo**: 200KB por foto
- **Tipo**: Foto profesional, preferiblemente con fondo neutro
- **Encuadre**: Centrado en el rostro

## Notas importantes

- Si no colocas las fotos, el sistema mostrará automáticamente **avatares generados** con las iniciales del nombre
- Las fotos deben ser **cuadradas** para verse mejor en el formato circular
- Puedes usar herramientas online como:
  - **Remove.bg** (https://www.remove.bg/) - Para quitar el fondo
  - **Squoosh** (https://squoosh.app/) - Para optimizar el tamaño
  - **Canva** (https://www.canva.com/) - Para crear fotos profesionales

## Vista previa

Cuando agregues las fotos, se mostrarán en formato circular en la página "Acerca de Nosotros":
- URL: http://localhost/tutorial%20laravel/sitio-facturacion/public/acerca-de

## Ejemplo de estructura final

```
public/images/team/
├── miguel-ayala.jpg    ✓
├── carlos-posada.jpg   ✓
├── gabo.jpg            ✓
└── README.md
```
